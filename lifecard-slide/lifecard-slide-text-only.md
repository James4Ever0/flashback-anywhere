# LifeCard

## 你的随身生活记录仪

开源 · 隐私优先 · 本地存储

---

## 什么是 LifeCard？

LifeCard 是一款**卡片式便携设备**，持续记录日常生活。

- 定时拍照
- 录音存档
- 数据同步

像随身携带的微型日记本。

---

## 产品形态

- **卡片大小**，可挂脖佩戴
- **10000mAh** 可扩展电池
- **ESP32-CAM** 主控芯片
- 轻量化外壳，随身无负担

---

## 项目目标

| 理念 | 说明 |
|------|------|
| 开源 | 固件、硬件、App 全部开放 |
| 尊重隐私 | 数据主权归用户 |
| 本地优先 | 可选不上云 |
| 可定制 | 配置文件 + 多种固件 |
| 安全耐用 | 可靠佩戴，长期运行 |

---

## 核心功能

- 定时拍照记录日常
- 音频录制
- USB / 蓝牙 / Wi-Fi 同步
- 两颗物理按钮：开关机、蓝牙配对
- LED 状态指示：电量、Wi-Fi、蓝牙、同步状态

---

## 硬件亮点

| 组件 | 规格 |
|------|------|
| 主控 | ESP32-CAM |
| 电池 | 10000mAh 可扩展 |
| 佩戴 | 轻量化挂脖外壳 |
| 交互 | 双按键 + 多色 LED |

---

## 软件生态

**PC 端**
- 照片导入、浏览
- 同步管理

**Android App**
- 移动端随时查看
- 照片管理与同步

**配置方式**
- 现阶段：配置文件
- 后续：App 可视化配置

---

## AI 能力

- **单张照片** → AI 生成描述
- **一天** → 生成日报总结
- **一周** → 周报回顾
- **一月** → 月度生活报告
- **对话式** → 向 AI 询问生活建议

---

## 存储与扩展

| 维度 | 选项 |
|------|------|
| 存储位置 | 本地 / 云端（可选）|
| AI 推理 | 本地模型 / 云端 API |
| 扩展功能 | 第三方 API：实时预览、录像…… |

---

## 现阶段目标用户

- 喜欢记录生活的技术爱好者
- 熟悉 Linux / Python
- 愿意编辑配置文件
- 优先选择本地存储，抗拒云端
- 想亲手折腾硬件
- AI 功能可有可无

---

## 下一阶段目标用户

- 普通生活记录者，不会编程
- 习惯云端存储
- 希望用 AI 获得生活洞察
- 开箱即用，无需配置
